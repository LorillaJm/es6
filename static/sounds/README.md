# Notification Sounds

Place your notification sound files here:

- `notification.mp3` - Default notification sound
- `notification-urgent.mp3` - Urgent/emergency notification sound

## Tips
- Keep files small (under 100KB recommended)
- Short sounds work best (0.5 - 2 seconds)
- You can find free notification sounds at:
  - https://notificationsounds.com/
  - https://freesound.org/
  - https://mixkit.co/free-sound-effects/notification/
